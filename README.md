# qfc-cli-morges
QFieldCloud Command Line Tool

## Install

```bash
git clone https://github.com/opengisch/qfc-cli-morges.git
cd qfc-cli-morges
pip install -r requirements.txt
cp .env.example .env
```

And configure the variables in the `.env` file

## Update

```bash
git pull
```

## Usage

```bash
python qfc_download.py
```
